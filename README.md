你可以使用熱鍵加速你的效率。

  - `Ctrl + C`: 複製文字。
  - `Ctrl + V`: 貼上複製內容。
  - `Ctrl + X`: 剪取文字。
  - `Ctrl + S`: 存檔。 你可以快速存檔，以及將`COMMIT_EDITMSG`存檔以提交至GitHub存放庫。
  - `Ctrl + Shift + S`: 另存新檔。
  - `Ctrl + Shift + P`: 尋找vscode設定。
  - `Ctrl + Shift + B`: 啟動專案建置工作。
  - `F5`: 啟動專案偵錯工作。
  - `Ctrl + F`: 尋找目標。
  - `Ctrl + H`: 尋找即取代目標。
  - `Ctrl + K` + `Ctrl + O`: 開啟資料夾。
  - `Ctrl + K` + `F`: 關閉資料夾。
  - `Shift + Enter`: 在Jupyter Notebook內執行單一個cell的直譯式偵錯或執行Markdown的編輯儲存。
